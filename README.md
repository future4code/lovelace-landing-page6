## O que funciona
 O usuário deve ser capaz de visualizar pelo menos duas seções diferentes na página
 O usuário deve ser capaz de visualizar ícones para redes sociais do empreendimento
 O usuário deve ser capaz ser encaminhado para a respectiva página ao clicar nos ícones
 Uma nova aba deve ser aberta para abrir as páginas de redes sociais
 A paleta de cores utilizada deve ser baseada na logomarca

 O usuário deve ser capaz de visualizar o conteúdo da página ao acessar por um smartphone
 O usuário deve conseguir encontrar todas as funcionalidades disponíveis quando se abre no navegador web
